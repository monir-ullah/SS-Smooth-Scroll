A lightweight plugin that enables smooth scrolling with adjustable speed using Lenis and GSAP.

== Description ==

SS Smooth Scroll is a WordPress plugin that implements smooth scrolling functionality using Lenis and GSAP libraries. It provides a simple settings interface to control scroll speed and enable/disable the feature.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/smooth-scrolls` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to Settings > General to configure the smooth scroll settings

== Frequently Asked Questions ==

= Can I adjust the scroll speed? =

Yes, you can adjust the scroll speed from the settings page. Values range from 0.5 (slow) to 2.0 (fast).
